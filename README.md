# Forest Rescue

Forest Rescue is a Unity third-person rescue game.

The game combines combat, rescue, and survival. The player controls Ryan to explore a forest area, defeat zombies near survivor tents, rescue survivors, and return to the safe house.

## Main Features

- Main menu with Start Game, Game Instructions, Game Background, Ranking List, and Complexity
- Easy Mode and Hard Mode
- Third-person player movement using Character Controller
- Player health system and damage feedback
- Zombie AI movement using NavMesh Agent
- Melee attack system using ArmSwing and AttackHitbox
- Weapon switching system with melee weapon and crossbow
- Crossbow ammo UI with limited arrows
- Health potion pickup and healing system
- Tent rescue system with survivor tracking
- Safe zone victory and game over conditions

## How to Play

- Use WASD to move
- Use the mouse to control the camera
- Click the left mouse button to attack
- Use the mouse wheel to switch weapons
- Press F near a tent to rescue the survivor after defeating the zombie near that tent
- Rescue all survivors and return to the safe house to win the game

## Unity Version

Unity 2022.3.62f3

## Final Goal

The player must rescue all survivors and return to the safe house.

If the player returns too early or the player's health becomes zero, the game shows Game Over.
